# The Last of Guss

Браузерный кликер, в котором игроки соревнуются, кликая по гусю, заражённому мутацией G-42.

## Стек

- React 19, TypeScript, Vite
- Tailwind CSS
- Zustand + TanStack Query
- React Router

## Особенности

- Раунды в реальном времени с обратным отсчётом
- Оптимистичное обновление счёта
- Таблица лидеров после завершения раунда
- Роль админа: может создавать раунды и смотреть статистику, но не может участвовать

## Запуск локально

```bash
npm install
npm run dev
```

Приложение запустится на http://localhost:5173

API проксируется на бэкенд-сервер.
